# 19 MYSQL性能优化


## 19.1 优化概述
如果MYSQL数据库需要进行大量的查询操作，那么就需要对查询语句进行优化。
如果连接MYSQL数据库用户很多，那么就需要对MYSQL服务器进行优化。

可以使用如下语句查询 MYSQL 数据库的性能：
```sql
SHOW STATUS LIKE 'value';
```
其中value取值如下：
1. Connections
2. Uptime
3. Slow_queries
4. Com_select
5. Com_insert
6. Com_delete
