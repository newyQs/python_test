#第5章 访问docker仓库
仓库（Repository）是集中存放镜像的地方，又分为公有仓库和私有仓库。

有时候把仓库与注册服务器（Registry）混淆。实际上注册服务器是存放仓库的具体服务器，一个注册服务器上可以有多个仓库，而每个仓库下面可以有多个镜像。
